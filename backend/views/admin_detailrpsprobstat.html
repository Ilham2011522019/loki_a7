Deskripsi Singkat Deskripsi Singkat Mata Kuliah Melalui mata kuliah ini mahasiswa dilatih agar mampu menerapkan konsep-konsep probabilitas dan statistik untuk menginterpretasikan data guna mendukung pemecahan masalah. Mahasiswa dibekali dengan pengetahuan dan keterampilan pengolahan data menjadi informasi dengan menggunakan statistik, sehingga dapat mengolah data secara efektif dan melakukan perhitungan secara kuantitatif bedasarkan prinsip-prinsip statistik, dan dapat menarik kesimpulan secara valid terhadap data yang diperoleh dari lapangan. dengan pendekatan case base methode.
